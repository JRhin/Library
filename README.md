## Visualization

Some useful functions used for visualization purposes.

* plot_decision_boundary():
  - Input:
    - model: a `torch.nn.Module`
    - X: a `torch.Tensor`
    - Y: a `torch.Tensor`
  - Ouput:
      Plots decision boundaries of model predicting on X in comparison to y.
